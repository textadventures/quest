Quest 5 lets you create text adventure games. It is free and open source. It is designed to be powerful, extensible and easy to learn. You can create games in any language - Quest has templates for English, French, German, Spanish, Dutch, Italian, Portuguese, Russian and other languages.

You can run Quest in a web browser, or you can download Quest for Windows.

<a href="https://textadventures.co.uk/quest" class="btn btn-purple" target="_blank">Use Quest in your browser</a>
[Download Quest for Windows](https://github.com/textadventures/quest/releases/download/v5.9/quest590.exe){: .btn .btn-green }

## Community and contributing

Quest is discussed on [GitHub Discussions](https://github.com/textadventures/quest/discussions) and [Discord](https://textadventures.co.uk/community/discord). If you find a bug or want to request a feature, [open an issue](https://github.com/textadventures/quest/issues).

Quest is completely open source - the source code and this documentation both live [on GitHub](https://github.com/textadventures/quest). See the [Developers](developers.html) page if you'd like to contribute, and [Other Guides](other_guides.html) for community-created tutorials and libraries.
